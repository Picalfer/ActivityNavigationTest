# Activity Navigation Test

[![Kotlin](https://img.shields.io/badge/Kotlin-1.8+-blue.svg?logo=kotlin)](https://kotlinlang.org)
[![Platform](https://img.shields.io/badge/Platform-Android-green.svg?logo=android)](https://www.android.com/)
[![Navigation](https://img.shields.io/badge/Feature-Activity_Navigation-blue.svg)](https://developer.android.com/guide/components/activities)

Тестовое приложение для изучения навигации между Activity в Android. Демонстрирует передачу данных и работу с результатами.

## Функциональность

- Навигация между тремя Activity
- Передача данных через Intent
- Получение результата из Activity (startActivityForResult)
- Различные способы завершения Activity

## Технологии

- Kotlin
- Activity navigation
- Intent data passing
- startActivityForResult
- ViewBinding
